# Cross Origin Resource Sharing (CORS)

- There are 2 way of Cross-origin request
  1. safe request
  2. All over other

## Safe Request

- Safe request is safe if it satisfies two condition:
  - [Safe-Method]: GET, POST or HEAD
  - [Safe-Headers]: Allow only custom headers are:
    1. `Accept`,
    2. `Accept-Language`,
    3. `Content-Language`,
    4. `Content-Type`
