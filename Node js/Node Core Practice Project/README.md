# Node Core Practice App
Complete learning of Node.js without Express. Covers:


✔ http module
✔ fs module
✔ path module
✔ url module
✔ streams
✔ buffers
✔ global variables
✔ manual routing
✔ JSON parsing
✔ handling POST data
✔ building controllers
✔ file-based database


## Start the server
node app.js


## Test Routes
GET / → Home
GET /create-user → User creation HTML page
POST /api/register → Add new user (from html)
GET /users → List users
POST /users → Add new user (JSON)
